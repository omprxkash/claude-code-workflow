# CLAUDE.md templates

Drop-in project brains. Copy one into your project root as `CLAUDE.md` — Claude Code reads it at the start of every session.

| File | What it's for |
|---|---|
| [`generic.md`](generic.md) | Any project — fill in the blanks |
| [`aios.md`](aios.md) | Personal AI OS setup — profile, 4 C's (Context / Connections / Capabilities / Cadence), skills table, self-updating failure log |
| [`trigger-dev.md`](trigger-dev.md) | Trigger.dev automation project — 9-step workflow, security rules, MCP tools, deploy checklist |

For domain-specific CLAUDE.md files pulled from real production repos, see [`system-prompts/`](../system-prompts/).
