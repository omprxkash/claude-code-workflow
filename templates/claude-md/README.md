# CLAUDE.md templates

Four real `CLAUDE.md` variants for different project types. Copy the one that fits, rename to `CLAUDE.md`, and drop it in your project root.

Full explanation: [../../docs/01-core-concepts/claude-md.md](../../docs/01-core-concepts/claude-md.md)

## Variants

| File | Use for |
|---|---|
| `skills-system.md` | A 26-skill automation architecture. Covers the Skills layer, subagents, build/review/QA loop, Modal webhooks, file structure, and operating principles. |
| `website-design.md` | Website recreation from a reference image. Screenshot-compare loop: generate → screenshot → compare → fix → repeat until match. |
| `executive-assistant.md` | Personal EA / second brain setup. Includes onboarding questions, context files structure, rules, decision log, and skills backlog. Use this as the setup prompt for a new EA project. |
| `wat-workflow.md` | WAT framework project (Workflows / Agent / Tools). Covers how to operate, self-improvement loop, file structure, and the workflow→tools separation. |

## Usage

Pick the closest one, copy it to your project root as `CLAUDE.md`, and adapt. Trim anything that doesn't apply. Target under 150 lines.
