# Why Claude Code

Claude Code is a terminal-first AI coding assistant that runs locally on your machine. Unlike a chatbot, it can read and edit files in your project, run scripts, call APIs, and do real work — all through natural language.

The core shift: instead of doing tasks manually or hand-wiring automations step by step, you describe what you want and the agent figures out the path, handles errors, and improves its own tools over time. Once you trust the output, you deploy the code it built to run on a schedule — without the agent present.

This repo is a practical reference for setting it up, understanding the key concepts, and working with the skills and workflows built on top of it.

Start with [install & setup](install-and-setup.md) if you haven't yet, or jump to [your first project](first-project.md) to see it in action.
