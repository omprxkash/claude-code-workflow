# 02 — Customization

How to make Claude Code behave the way you want — custom commands, skills, hooks, and agents. Start with slash commands, then skills, then subagents.

| File | What it covers |
|---|---|
| [slash-commands.md](slash-commands.md) | Built-in commands and how to write your own as `.md` files |
| [skills.md](skills.md) | Reusable capability bundles — what they are, how they activate, how to build one |
| [hooks.md](hooks.md) | Scripts that run before or after tool calls (format on save, lint, notify, etc.) |
| [subagents.md](subagents.md) | Isolated workers with scoped tools — the four real agents and the design+build loop |
| [agent-teams.md](agent-teams.md) | Running multiple agents in formation |
| [statusline.md](statusline.md) | Custom terminal status line |

Working agent files and CLAUDE.md templates are in [`templates/`](../../templates/).
